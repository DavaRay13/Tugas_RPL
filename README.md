🚀 Aplikasi Login Keren (Frontend Only)

Halo bro! Selamat datang di repository Aplikasi Login Keren.

Ini adalah project simulasi UI/UX untuk fitur otentikasi (Login & Register) yang dibangun tanpa database / backend. Semua data user yang didaftarkan akan disimpan sementara di dalam localStorage browser pengguna.

Project ini cocok banget buat lu yang mau belajar manipulasi DOM, state management sederhana pakai Vanilla JavaScript, dan styling pakai Tailwind CSS.

✨ Fitur Keren di Sini

Login & Register Flow: Alur pendaftaran dan masuk yang mulus dalam satu halaman (SPA feel).

Glassmorphism UI: Desain UI transparan dan modern kayak kaca.

Custom Toast Alerts: Notifikasi pop-up moderen (bukan alert() bawaan browser yang kaku).

Confetti Effect: Hujan kertas warna-warni pas user berhasil login (dibuat murni pakai JS + CSS).

No Build Tools Required: Tinggal klik buka, langsung jalan!

🛠️ Teknologi yang Dipakai

HTML5 (Semantik)

Tailwind CSS (via CDN - buat styling super cepat)

Vanilla JavaScript (buat logika form, localStorage, dan animasi)

🚀 Cara Jalanin Project Ini (Super Gampang)

Karena ini cuma frontend, lu nggak perlu repot-repot install Node.js, NPM, atau setup database.

Clone repository ini ke laptop lu:

git clone [https://github.com/username-lu/nama-repo-lu.git](https://github.com/username-lu/nama-repo-lu.git)


Buka folder project-nya.

Klik dua kali di file index.html (atau buka pakai ekstensi Live Server di VS Code biar lebih enak).

Selesai! Cobain daftar akun ngasal dan cobain login.
